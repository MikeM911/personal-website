jokeM Mobile Application

The joke mobile application was utilized with JokeApiV2 API as the source for varieties of jokes to be fetched and output a joke. The joke app was designed with Flutter and Dart as the main languages to construct the frontend design. Users can select a joke based on the provided categories such as Programming, Technology, Christmas theme, etc. Meanwhile, the users have the option to blacklist some options based on what types of jokes users would like to see. Once a joke has been selected, it is then saved onto a local database (SQLite). Also, some features of the app allows the user to fetch another joke or save it onto their favorites tab.
